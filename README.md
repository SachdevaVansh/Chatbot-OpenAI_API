# Chatbot-OpenAI_API
This repo is for creating a Telegram Chatbot

## Prerequistes :
- python 3.7 
- A telegram account
- an OpeAI account for API key

## How to run the code :
'''bash
conda create -n telebot python=3.7 -y
'''

'''bash
conda activate telebot
'''

'''bash
pip install -r requirements.txt
'''
